# Engineer Profile Chatbot

An AI-powered chatbot that can discuss Leandro R.'s engineering background,
skills, and projects. Uses the HF Inference API with the engineer profile
as system context.

## Setup

Set the `HF_TOKEN` secret in your Space settings.
Optionally set `MODEL_ID` to use a different model (default: `Qwen/Qwen2.5-72B-Instruct`).
